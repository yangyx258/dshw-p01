## 数据来源

- 股票行情：`akshare.stock_zh_a_daily()`，后复权，日度，时间范围为 2020-01-01 至当前可得最新交易日。
- 市场指数：`akshare.stock_zh_index_daily()`，包含沪深 300（000300）和上证指数（000001）。沪深 300 作为 CAPM 市场基准；上证指数用于补充观察 A 股整体大盘走势。
- 宏观指标：`akshare`，包含 CPI 同比增速和 M2 同比增速。CPI 反映通胀压力，M2 反映货币流动性，两者都可能影响股票估值和风险偏好；后续宏观关系图以 M2 同比增速为主分析指标。
- 财务数据：`akshare.stock_financial_analysis_indicator()` 年度财务指标，包括营业收入增长率、销售净利率和 ROE。

## 存储方式

- 基础：CSV。CSV 可读性强、跨软件兼容好，适合课程作业提交和小规模数据交换；不足是缺少严格类型约束，文件体积较大，读取大数据时效率较低。
- 进阶：Parquet。选择 Parquet 是因为本作业以表格型时间序列数据为主，Parquet 支持列式读取、保留 Schema，适合后续只读取少数变量进行分析。
